The mathematics behind CSSFINDER

A quantum state of a finite-dimensional system is described by a density matrix \(\rho\), a positive semi-definite operator on a Hilbert space \(\mathcal{H}\) with unit trace: \(\rho \geq 0\), \(\operatorname{Tr}[\rho] = 1\). Pure states satisfy \(\rho^2 = \rho\); mixed states satisfy \(\operatorname{Tr}[\rho^2] < 1\).

For a bipartite system with subsystems \(A\) and \(B\) of dimensions \(d_A\) and \(d_B\), the full Hilbert space is \(\mathcal{H} = \mathcal{H}_A \otimes \mathcal{H}_B\).

A state \(\rho_{AB}\) is separable if it can be written as a convex combination of pure product states:

$$\rho_{\mathrm{sep}} = \sum_i p_i \, \rho_i^A \otimes \rho_i^B, \qquad p_i \geq 0,\; \sum_i p_i = 1.$$

States that cannot be written in this form are entangled. The set of separable states \(\mathcal{S}\) is convex and compact, which makes it amenable to geometric optimization methods. For multipartite systems the same definition extends: a state is fully separable if it decomposes into a mixture of fully product states \(\rho_i^{(1)} \otimes \cdots \otimes \rho_i^{(N)}\).

The Hilbert-Schmidt distance between two density matrices \(\rho_0\) and \(\rho_1\) is defined via the Hilbert-Schmidt (Frobenius) inner product on the space of operators:

$$D^2(\rho_0,\,\rho_1) = \operatorname{Tr}\!\left[(\rho_0 - \rho_1)^2\right].$$

It is invariant under unitary conjugation — \(\operatorname{Tr}[(UAU^\dagger)(UAU^\dagger)^\dagger] = \operatorname{Tr}[AA^\dagger]\) — which makes it well-suited to quantum state space geometry. While it does not satisfy all axioms of a formal entanglement measure, it provides a computationally efficient proxy: the closer \(\rho_0\) is to \(\mathcal{S}\), the less entangled it is in a geometric sense.

Closest Separable State

The Closest Separable State (CSS) of a density matrix \(\rho_0\) is the separable state \(\rho_1^* \in \mathcal{S}\) that minimises the Hilbert-Schmidt distance:

$$\rho_1^* = \operatorname*{arg\,min}_{\rho_1 \in \mathcal{S}} \; D^2(\rho_0,\,\rho_1).$$

The squared distance \(D^2(\rho_0, \rho_1^*)\) quantifies how far \(\rho_0\) lies from the separable set. For a maximally entangled \(d\)-dimensional bipartite state \(|\phi_d\rangle = \frac{1}{\sqrt{d}}\sum_{i=0}^{d-1}|i,i\rangle\), the CSS is a Werner state and the exact value is known analytically:

$$D^2\!\left(|\phi_d\rangle\langle\phi_d|,\,\mathcal{S}\right) = \frac{d-1}{d+1}.$$

Entanglement Witnesses

Given the CSS \(\rho_1^*\), one can construct an entanglement witness operator:

$$W_{\rho_0} = (\rho_0 - \rho_1^*) - \max_{|\phi\rangle \in \mathcal{S}} \langle\phi|(\rho_0 - \rho_1^*)|\phi\rangle \cdot \mathbf{I}.$$

Any separable state \(\sigma\) satisfies \(\operatorname{Tr}[W_{\rho_0}\,\sigma] \geq 0\), while \(\operatorname{Tr}[W_{\rho_0}\,\rho_0] < 0\) certifies entanglement of \(\rho_0\).

Gilbert's algorithm is a classical method for finding the nearest point in a convex set to a given query point. CSSFINDER adapts it to the space of density matrices, iteratively building a separable state that converges toward \(\rho_1^*\).

Preselection criterion

A random pure product state \(\rho_2\) is a useful candidate only if it lies on the same side of the current tangent hyperplane as \(\rho_0\). This is checked via:

$$\operatorname{Tr}\!\left[(\rho_2 - \rho_1)(\rho_0 - \rho_1)\right] > 0.$$

Geometrically, this ensures the angle between \(\rho_0 - \rho_1\) and \(\rho_2 - \rho_1\) does not exceed \(\pi/2\). Candidates failing this test are discarded without an expensive minimisation step.

Algorithm steps

  • 1 Increment the trial counter \(c_t\). Draw a random pure product state \(\rho_2 = |\psi\rangle\langle\psi|\) with \(|\psi\rangle = |\psi^{(1)}\rangle \otimes \cdots \otimes |\psi^{(N)}\rangle\), sampled uniformly with respect to the Hilbert-Schmidt measure.
  • 2 Apply the preselection test. If \(\operatorname{Tr}[(\rho_2-\rho_1)(\rho_0-\rho_1)] \leq 0\), restart from Step 1.
  • 3 Symmetrize \(\rho_2\) with respect to any discrete or continuous symmetries shared by \(\rho_0\) that preserve separability: \(\rho_2 \leftarrow \frac{1}{k}\sum_{z=0}^{k-1} U^z \rho_2 (U^\dagger)^z\).
  • 4 Minimize \(D^2(\rho_0,\, p\rho_1 + (1-p)\rho_2)\) over \(p \in [0,1]\). The optimal mixing parameter is: \(p^* = \operatorname{clip}\!\left(\frac{\operatorname{Tr}[(\rho_0-\rho_2)(\rho_1-\rho_2)]}{\operatorname{Tr}[(\rho_1-\rho_2)^2]},\, 0,\, 1\right).\)
  • 5 Update the current approximation: \(\rho_1 \leftarrow p^*\rho_1 + (1-p^*)\rho_2\). Record the new squared distance \(D^2(\rho_0,\rho_1)\) and increment the success counter \(c_s\).
  • 6 Halt when \(c_s\) reaches the iteration limit or \(D^2(\rho_0,\rho_1)\) falls below the convergence threshold. Otherwise return to Step 1.

Convergence

The algorithm is guaranteed to converge because \(\mathcal{S}\) is convex and each accepted step strictly decreases \(D^2\). In practice the convergence follows an empirical scaling law: defining \(y_{c_s} = |\ln[D^2_{c_s} - a]|^b\) where \(a \approx \lim_{c_s\to\infty} D^2\) is the estimated true HSD, the plot of \(y_{c_s}\) versus \(c_s\) is nearly linear, enabling extrapolation to tight upper bounds with far fewer iterations than brute-force sampling.

Bell state \(|\Phi^+\rangle\)

The maximally entangled two-qubit state:

$$|\Phi^+\rangle = \frac{1}{\sqrt{2}}\left(|00\rangle + |11\rangle\right), \qquad D^2 = \frac{1}{3}.$$

GHZ state \(|\mathrm{GHZ}\rangle\)

The three-qubit Greenberger-Horne-Zeilinger state:

$$|\mathrm{GHZ}\rangle = \frac{1}{\sqrt{2}}\left(|000\rangle + |111\rangle\right).$$

GHZ entanglement is highly non-robust: loss of any one qubit collapses the remaining two qubits into a fully mixed, separable state. Convergence to the CSS is substantially slower than for the Bell state.

W state \(|\mathrm{W}\rangle\)

The three-qubit W state:

$$|\mathrm{W}\rangle = \frac{1}{\sqrt{3}}\left(|001\rangle + |010\rangle + |100\rangle\right), \qquad D^2 \approx 0.4322.$$

Unlike GHZ, W-state entanglement is robust under single-qubit loss — the reduced two-qubit state remains entangled. This structural difference is reflected in the convergence behaviour of the algorithm.
